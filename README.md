# Бот поздравлений в мессенджере МАХ с использованием ИИ  @digitalhach_bot.

Бот для автоматизации создания и отправки корпоративных поздравлений: пользователь выбирает повод и стиль → бот через GigaChat генерирует текст и открытку → показывает превью → отправляет в MAX или на email. 
Доступна функция отложенной отправки. 

## Функционал

- 🤖 Генерация персонализированных текстов поздравлений через GigaChat
- 🖼️ Создание открыток с помощью Kandinsky
- 📧 Отправка по email и в MAX Messenger (для пользователей запустивших бота. Корпоративный бот подразумевает, что у сотрудников будет активирован бот).
- ⏰ Отложенные поздравления по расписанию
- **История** отправленных поздравлений (`/history`)
- Понятная обработка ошибок (GigaChat недоступен, неверный email)
- Перегенерация **отдельно** текста и картинки
- Учет корпоративного контекста при поздравлениях
- Динамический календарь в кнопках (весной не будет отображаться поздравить с Новым годом)

## Технологии
- Python 3.11 / FastAPI
- SQLite / PostgreSQL
- GigaChat API
- Render.com

## P.S.

Бот может не работать в моменте просмотра, так как токен бота принадлежит Мэрии Казани и может быть отключен или перенесен.
